플랫폼 이름

by 치약좋지

Linux Vulnerability Diagnosis Automation Platform


본 플랫폼은 KISA 주요정보통신기반시설 기술적 취약점 분석 상세 가이드(2026) 를 기준으로 설계된
엔터프라이즈 리눅스 취약점 진단 자동화 플랫폼입니다.

수동 점검 중심의 비효율적인 운영 방식을 개선하고, 표준화된 정책 기반 진단 체계를 자동화하여
조직 내 보안 수준을 일관되게 유지할 수 있도록 지원합니다.

단일 서버부터 대규모 인프라 환경까지 확장 가능한 보안 점검 서비스를 제공합니다.



Single Server Assessment

IP 입력 기반 실시간 취약점 자동 진단.
KISA 표준 항목 기반 정밀 점검 수행.

Bulk Server Inspection

CSV 업로드 기반 다수 서버 일괄 분석.
운영 환경에 최적화된 대규모 자동 점검 수행.

Automated Reporting

진단 결과 자동 정리 및 Word 보고서 생성.
감사 대응 및 문서화 지원.

CVE Intelligence Integration

설정 취약점 + 공개 취약점 동시 분석.
정책 기반 진단과 실시간 위협 인텔리전스 결합.
